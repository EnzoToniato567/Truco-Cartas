# Truco-Cartas

Um jogo interativo de Truco desenvolvido em HTML, CSS e JavaScript.

## Descrição

Este projeto implementa o clássico jogo de cartas Truco com interface web interativa, permitindo que os jogadores desfrutem de uma experiência divertida e desafiadora diretamente no navegador.

## Tecnologias Utilizadas

- **HTML5** - Estrutura e marcação semântica
- **CSS3** - Estilização e layout responsivo
- **JavaScript** - Lógica do jogo e interatividade
- **GitHub** - Controle de versão e repositório
- **Visual Studio Code** - Editor de código
- **Live Server Extension** - Servidor local para desenvolvimento
